Features

Features:

  • Discover music, artists and tracks
  • No need to install Spotify or create any accounts.
  • Search directly from Chrome addressbar (omnibox)
  • Right click on selected text and search
  • Auto-suggest as you type
  • Shows you the HTTP and EMBEDDING code so that you can share the music on your blog, website, twitter or Facebook...
  • Automatically load more results on-demand (saves network bandwidth)
  • Spotify compliant look and feel
  • All the codes and graphics are visible as an Open Source project

If you want more features, please let me know on the Facebook page of Spotify Search.

This extension and all the graphical design elements (which include icons, color combinations, Spotify Search logo except the pages that are loaded directly from Spotify) are designed, coded and tested by Alex Hanif. The code of the project is available for free on Github.

The story behind this extension

There is a story behind this extension. Just like most people who use Spotify (and this extension) I love music. I happen to be a web architect living and working in Stockholm so naturally when I was called by a head hunter recruiting for Spotify I didn't say no.

But the interview didn't go very well, I have had a habid of using my hand a lot while talking. Over the interview I got too excited about something and with a move of my hand I spilled the coffee over the table, my resume, interviewer and a little bit on the wall behind her! (ture story) Naturally I din't get further in the recruitment process but I don't give up easily.

I applied once more in April 2012 but Spotify HR department thought I'm more of a designer rather than coder (actually I'm both, but I've programmed for the last 14 years) but unfortunately most of my commercial code is written for high security proprietary projects. So I decided to write some code and the easter holiday of 2012 was the birth of this extension. I just wanted to prove that I can write clean and nice code with usability and aesthetics in mind. I didn't get the job at Spotify but I got 200+ users in a few weeks! People were actually using the demo code! That made me keep working on this extention to make it useful and here we are.

If you like it, please drop me a line in the Facebook page. It really warms up my heart. I'm getting paid very well at my company and this extension is something I do for fun so there's no need to donate. Just some feedback would be nice as I always appreciate what users feel about my products.
