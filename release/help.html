How to search?

There are basically 3 ways to search with this extension:

1. Type in the address bar

The easiest one is to type "Spotify" in the addressbar (1):

And then type the search terms and press Enter. The extension tries to guess what you are typing and makes up to 5 suggested track names (2). You can choose any of those guesses or directly go to the main page of the Spotify Search extension (1).

2. Search the selected text

You can easily search for whatever text you can select around the internet. Just select some text (1) and right-click with the mouse and choose "Search Spotify" (2) from the menu.

3. Main search window

This is the most feature-full method of search. For example the auto-complete feature in the address bar only suggests track names. But when you type in the search bar of the Spotify Search extension page (2), you get suggestions for artists and albums as well (3). To see the main page, simply click onthe Spotify Search icon at the top right corner of your Chrome (1).

Search results

The results page isr eally powerful. You can access more information about most items on screen. Just hover your mouse over any artist name (1), album name (2) or track name (3) to go to a page that gives you more information.

The stars show popularity on a scale of 0 (all grey) to 10 (all yellow).
